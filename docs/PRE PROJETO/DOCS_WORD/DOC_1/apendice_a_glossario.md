# Apendice A: Glossario

| **Termo**           | **Definicao**                                                 |
| ------------------- | ------------------------------------------------------------- |
| SINAPI              | Sistema Nacional de Pesquisa de Custos e Indices (Caixa/IBGE) |
| SICRO               | Sistema de Custos Referenciais de Obras (DNIT)                |
| RSMeans             | Base de dados de custos da Gordian (referencia internacional) |
| BDI                 | Bonificacao e Despesas Indiretas (markup sobre custo direto)  |
| INCC                | Indice Nacional de Custo da Construcao (FGV)                  |
| AACE                | Association for the Advancement of Cost Engineering           |
| Composicao Unitaria | Decomposicao servico em insumos, MO e equipamentos            |
| Score de Confianca  | Metrica 0-100 indicando certeza do custo estimado             |
| RLS                 | Row Level Security (seguranca por linha no BD)                |
| PLG                 | Product-Led Growth                                            |
| MRR                 | Monthly Recurring Revenue                                     |
| ARPU                | Average Revenue Per User                                      |
| LTV                 | Lifetime Value                                                |
| CAC                 | Customer Acquisition Cost                                     |
