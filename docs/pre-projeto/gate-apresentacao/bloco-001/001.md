## Objetivo

Construir um motor de orçamentação inteligente para obras de montagem elétrica industrial, substituindo o Arquimedes e as planilhas Excel. O produto inclui um dimensionador elétrico (cálculo técnico por NBR 5410) e um motor de estimativa com assemblies parametrizáveis, que gera propostas comerciais completas.

## Contexto

### O que já existe

- **PWC módulo orçamentos**: 18 telas funcionais, ~9.000 linhas, pricebook multi-contexto, WBS hierárquica (CHAPTER/PACKAGE/ACTIVITY), sistema de revisões, catálogos de materiais/MO/equipamentos/impostos/markup
- **App Monte Carlo**: `montecarlo.conradomatos.dev` — simulação Beta-PERT funcionando
- **Parser BC3**: Python e JS funcionando
- **Spec 0 (Destilação)**: Em andamento — 7 fontes de dados sendo extraídas (HOLLOS, MURILO, Concept, Arquimedes, SINAPI-PR, RSMeans, Siemens)

### O que NÃO existe (gaps críticos)

1. **Assemblies parametrizáveis** — composições inteligentes (material + MO/equipe + equipamento + rendimento + fatores de ajuste). Gap mais crítico
2. **Dimensionador elétrico** — cálculo de bitola, infra por trecho, alimentadores (NBR 5410)
3. **Fatores de ajuste** — dificuldade, produtividade, regional, escala
4. **Faixa de preço** — mínimo / provável / máximo por item
5. **Motor paramétrico** — cálculo é soma simples hoje, sem modelagem
6. **IA** — leitura de escopo, auditoria de consistência, sugestão de composições
7. **Importação SINAPI/RSMeans** integrada
8. **Cotações (RFQ)** — solicitar e comparar propostas de fornecedores
9. **Histórico / benchmarking** — comparar orçamento atual com anteriores

### Incluído

- Dimensionador elétrico completo (6 módulos)
- Motor de estimativa com assemblies parametrizáveis
- Base de custos alimentada pelo Spec 0 (destilação)
- Cotações (RFQ)
- Histórico / benchmarking
- Integração Monte Carlo
- Todos os custos indiretos
- Importação via Excel em todos os módulos
- Geração automática de lista de materiais com rastreabilidade
