**Parâmetros globais (nível empresa/sistema)** — mudam raramente, valem pra todos os orçamentos até alguém alterar. São configurações do "ambiente":

- Perfis de encargos sociais (Lucro Real, Presumido, Simples)
- Tabela salarial base (dissídio vigente)
- Alíquotas de impostos por UF (ICMS, ISS)
- Taxas de BDI padrão (ADM 10%, Lucro 15%, etc.)
- Catálogo de equipamentos com custo/dia
- Itens de canteiro com custo padrão
- Fórmula de reajuste de commodities (LME, câmbio)
