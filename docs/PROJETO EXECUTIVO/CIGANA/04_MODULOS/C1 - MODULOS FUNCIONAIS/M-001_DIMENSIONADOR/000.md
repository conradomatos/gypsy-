### EPIC E-002: Dimensionador Elétrico

Produto separado. Ferramenta de cálculo técnico NBR 5410. Recebe dados de carga, calcula bitolas e dimensiona infra, gerando lista de materiais quantificada por área. Entrada principal: lista de motores do cliente via Excel. Saída: lista de materiais → alimenta E-003.

**Fases:** PH-1 (Força + Infra), PH-3 (Integração), PH-5 (Iluminação + Painéis + Sensores)
>[!coment] Conrado
>Nao e que ele seja produto separado, mas é um produto fora da classificacao de orcamentos, porem vamos colocar ele dentro.
