# MC-001.2 — Alimentação Pessoal

## O que é

Custo de alimentação dos trabalhadores no site da obra.
Entra no MC-1 como componente do custo/hora — não é BDI, é custo direto da MO.

## Componentes

| Item | Unidade | Default Concept | Observação |
|------|---------|-----------------|------------|
| Café da manhã | R$/dia | 8,00 | Só se obra com hospedagem |
| Almoço | R$/dia | 25,00 | Sempre (PAT ou refeitório) |
| Jantar | R$/dia | 20,00 | Só se regime com pernoite ou hora extra |
| Ceia/lanche | R$/dia | 5,00 | Se turno noturno |

## Fórmula

```
custo_alimentacao_dia = cafe + almoco + jantar + ceia
custo_alimentacao_mes = custo_alimentacao_dia × dias_uteis_mes
```

## Regras

- Alimentação **não incide** INSS, FGTS, IRRF (quando fornecida via PAT)
- É parâmetro do **projeto** (override do global) — cada obra tem condições diferentes
- Café e jantar dependem do flag `hospedagem > 0` (se hospeda = 3 refeições, se local = só almoço)
- Ceia depende do flag `turno_noturno = true`

## No código

Parâmetro em `ProjectParameters.meals`:
```typescript
interface MealCosts {
  breakfast: number;   // café
  lunch: number;       // almoço
  dinner: number;      // jantar
  snack: number;       // ceia/lanche noturno
}
```
