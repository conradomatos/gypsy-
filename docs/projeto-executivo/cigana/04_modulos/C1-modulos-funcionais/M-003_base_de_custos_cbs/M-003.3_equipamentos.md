# M-003.3 — Equipamentos

Catálogo de equipamentos e ferramentas com custo operacional.

## O que contém

- Guindastes (30/60/100 ton)
- Muncks (12-43 ton, com/sem plataforma, com/sem diesel)
- Plataformas elevatórias (tesoura, articulada 8-22m)
- Retroescavadeira, Bob Cat, Betoneira
- Geradores (1-40 KVA)
- Andaimes (m³ montado/desmontado)
- Ferramentas (furadeira, esmerilhadeira, rosqueadeira, soldadora)
- Rádio comunicação, máscara de fuga, caixas de ferramentas

## Estrutura de dados

Por equipamento:
- Código, descrição, categoria
- Unidade (DIA/MÊS/HORA)
- Valor unitário (locação de mercado OU calculado por depreciação)
- Consumo combustível/hora (quando aplicável)
- Operador necessário (sim/não)

## Cálculo de custo de propriedade (quando não é locação)

Fórmula: depreciação 60 meses + remuneração investimento 1,5%/mês + manutenção 15%/ano + combustível/hora
