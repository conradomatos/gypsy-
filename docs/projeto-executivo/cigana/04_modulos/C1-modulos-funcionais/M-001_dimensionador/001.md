## Cenários de uso (User Stories)

### Dimensionador Elétrico

1. Como orçamentista, quero importar a lista de motores do cliente via Excel para não precisar cadastrar motor por motor.
2. Como orçamentista, quero que o sistema calcule automaticamente a bitola do cabo de potência e comando para cada motor usando NBR 5410.
3. Como orçamentista, quero cadastrar os trechos de infra e associar quais motores passam por cada trecho para o sistema calcular a ocupação e dimensionar o leito/eletrocalha.
4. Como orçamentista, quero que o sistema gere automaticamente as terminações (sealtub, box, prensa-cabo) para não esquecer nenhum item.
5. Como orçamentista, quero lançar luminárias e tomadas por área para gerar a lista de materiais de iluminação com infra.
6. Como orçamentista, quero que o sistema dimensione o alimentador do painel e o barramento a partir da soma de cargas dos motores.

>[!coment] Conrado
>Talvez seja necessário reavaliar o pipeline atual. - 1 lançar áreas, equipamentos etc. (WBC) depois lançar os dados de motores/iluminação/tomadas/infiras por áreas/equipamentos em dimensionamento. É a sequencia logica, primeiro definir onde está, fase, área etc. depois ja lança na área certa.
### Motor de Estimativa

7. Como orçamentista, quero criar composições analíticas (assembly) com materiais, MO e equipamentos, cada uma com rendimento unitário e fatores de ajuste.
8. Como orçamentista, quero aplicar fatores de dificuldade, produtividade e regional nas composições para refletir condições reais da obra.
>[!coment] Conrado
>Deve ser possivel aplicar de forma selecionada, individual ou por categoria etc. como usuario preferir
9. Como orçamentista, quero que cada composição tenha faixa de preço (mínimo/provável/máximo) para alimentar a simulação Monte Carlo.
10. Como orçamentista, quero importar composições SINAPI e RSMeans como base e customizar rendimentos e preços.
>[!coment] Conrado
>Precisa tomar cuidado aqui, a intencao é que nao precise mexer nisso no dia dia porque iremos entregar um banco de composicoes mais completa possivel, se for alterar algo, nao pode ser qualquer um.
11. Como orçamentista, quero organizar o escopo por área/local (WBS) e vincular composições a cada nó.
12. Como orçamentista, quero lançar itens que vieram do dimensionador e itens manuais (andaime, lift, serviços avulsos) no mesmo orçamento.
>[!coment] Conrado
>Precisa tomar cuidado aqui
13. Como orçamentista, quero solicitar cotações de fornecedores e comparar propostas para aplicar o melhor preço no orçamento.
14. Como orçamentista, quero comparar o custo unitário dos itens do orçamento atual com projetos anteriores para identificar outliers.

### Saídas

15. Como orçamentista, quero rodar simulação Monte Carlo sobre o orçamento para ver P10/P50/P90 do custo total.
>[!coment] Conrado
>Calma la, orcamentista nao tem nem ideia do que é monte carlo, mal engenheiro tem, isso nao é para qualquer um usar.
16. Como orçamentista, quero gerar proposta completa em PDF no template da Concept com um clique.
17. Como diretor comercial, quero comparar orçado vs. realizado de projetos anteriores para calibrar margens futuras.

### IA

18. Como orçamentista, quero que a IA leia o edital do cliente e sugira os itens do orçamento para acelerar o lançamento.
19. Como orçamentista, quero que a IA audite meu orçamento antes de enviar a proposta para identificar preços fora da faixa e itens faltantes.
