### EPIC E-006: Inteligência Artificial

>[!coment] Conrado
>Fazer Briefing - ia e fase futura

Camada de IA sobre o motor de estimativa. 3 capacidades: leitura de escopo, auditoria de consistência, sugestão de composições. Usa Claude API.

**Fase:** PH-7 (depende de PH-2 e PH-3)

#### FEATURE F-050: Leitura de Escopo

IA lê PDF/edital do cliente e extrai lista de serviços para o orçamentista revisar.

**STORY S-400:** Como orçamentista, quero que a IA leia o edital do cliente e sugira os itens do orçamento.

| ID    | Descrição                                                                                      | Tipo | Esforço |
| ----- | ---------------------------------------------------------------------------------------------- | ---- | ------- |
| T-500 | Implementar upload e parse de PDF/edital do cliente                                            | Task | M       |
| T-501 | Implementar prompt de extração: identificar serviços, quantidades, especificações no documento | Task | L       |
| T-502 | Implementar match: serviço extraído → assembly mais provável no catálogo                       | Task | L       |
| T-503 | UI: tela de review com itens extraídos, assembly sugerida, confiança, aceitar/rejeitar/editar  | Task | L       |


#### FEATURE F-052: Geração de Faixa de Preço

IA gera automaticamente faixa min/provável/máx para assemblies sem dados históricos.

**STORY S-420:** Como orçamentista, quero que a IA sugira faixas de preço quando não tenho dados suficientes.

| ID    | Descrição                                                                            | Tipo | Esforço |
| ----- | ------------------------------------------------------------------------------------ | ---- | ------- |
| T-520 | Implementar prompt de estimativa: dado o serviço + região + condições, sugerir faixa | Task | L       |
| T-521 | Implementar validação cruzada: faixa sugerida vs. SINAPI vs. histórico da Concept    | Task | M       |
| T-522 | UI: indicador "estimado por IA" com nível de confiança                               | Task | S       |
