# MC-001.4 — Fórmula Consolidada do Custo/Hora

## Pseudocódigo do engine

```typescript
function calculateHourlyCost(
  role: LaborRole,
  charges: SocialChargeProfile,
  params: ProjectParameters
): HourlyCostResult {

  // 1. Remuneração bruta
  const hazardPay = role.hasHazardPremium ? role.baseSalary * 0.30 : 0;
  const unhealthyPay = params.unhealthinessGrade !== 'none'
    ? MINIMUM_WAGE * UNHEALTHINESS_RATES[params.unhealthinessGrade]
    : 0;
  const grossPay = role.baseSalary + hazardPay + unhealthyPay;

  // 2. Encargos sociais (Grupo A)
  const groupA = grossPay * (
    charges.groupA.inssPatronal
    + charges.groupA.fgts
    + charges.groupA.rat
    + charges.groupA.thirdParty
    + charges.groupA.seconci
  );

  // 3. Provisões trabalhistas (Grupo B)
  const thirteenth   = grossPay * charges.groupB.thirteenthSalary;
  const vacation     = grossPay * charges.groupB.vacation;
  const vacationBon  = grossPay * charges.groupB.vacationBonus;
  const dsr         = grossPay * charges.groupB.dsr; // 0 para mensalista
  const groupB = thirteenth + vacation + vacationBon + dsr;

  // 4. Incidência de A sobre B (Grupo C)
  const groupARate = charges.groupA.inssPatronal + charges.groupA.fgts
    + charges.groupA.rat + charges.groupA.thirdParty + charges.groupA.seconci;
  const groupC = groupB * groupARate;

  // 5. Rescisórios (Grupo D)
  const groupD = grossPay * (
    charges.groupD.noticePeriod
    + charges.groupD.fgtsOnNotice
    + charges.groupD.fgtsPenalty
  );

  // 6. Benefícios (não incidem encargos)
  const meals = (params.meals.breakfast + params.meals.lunch
    + params.meals.dinner + params.meals.snack) * params.workingDaysMonth;
  const transport = params.transportDaily * params.workingDaysMonth;
  const lodging = params.lodgingDaily * params.workingDaysMonth;
  const epiTools = role.epiMonthly + role.toolsMonthly;
  const examsInsurance = role.examsMonthly + role.lifeInsuranceMonthly;

  // 7. Custo total mensal
  const totalMonthly = grossPay + groupA + groupB + groupC + groupD
    + meals + transport + lodging + epiTools + examsInsurance;

  // 8. Custo/hora
  const hoursMonth = params.hoursPerMonth; // 220 (mensalista) ou 180 (horista)
  const hourlyCostWithoutBdi = totalMonthly / hoursMonth;

  return {
    hourlyCostWithoutBdi,
    hourlyCostWithBdi: hourlyCostWithoutBdi / (1 - params.bdiRate),
    overtime50: hourlyCostWithoutBdi * 1.50,
    overtime100: hourlyCostWithoutBdi * 2.00,
    nightShift: hourlyCostWithoutBdi * 1.20,
    breakdown: {
      grossPay, hazardPay, unhealthyPay,
      groupA, groupB, groupC, groupD,
      meals, transport, lodging, epiTools, examsInsurance,
      totalMonthly, hoursMonth
    }
  };
}
```

## Exemplo numérico — Eletricista Oficial, Lucro Real, obra local

| Componente | Cálculo | R$/mês |
|-----------|---------|--------|
| Salário base | — | 3.500,00 |
| Periculosidade 30% | 3.500 × 0,30 | 1.050,00 |
| **Remuneração bruta** | | **4.550,00** |
| Grupo A (37,80%) | 4.550 × 0,378 | 1.719,90 |
| Grupo B mensalista (20,83%) | 4.550 × 0,2083 | 947,77 |
| Grupo C (A×B = 7,87%) | 4.550 × 0,0787 | 358,09 |
| Grupo D (7,65%) | 4.550 × 0,0765 | 348,08 |
| Almoço (25/dia × 22) | | 550,00 |
| EPI + Ferramentas | 180 + 50 | 230,00 |
| Exames + Seguro | 25 + 15 | 40,00 |
| **Total mensal** | | **8.743,84** |
| **Custo/hora (÷220h)** | | **R$ 39,74** |
| **Custo/hora com BDI 26,76%** | 39,74 / (1 - 0,2676) | **R$ 54,26** |

> **Multiplicador final:** 8.743,84 / 3.500 = **2,50×** o salário base
> Com BDI: **3,41×** o salário base → preço de venda da hora
