# MC-001.3 — Adicionais CLT

## Periculosidade

- **Percentual:** 30% sobre o salário base (não sobre outros adicionais)
- **Base legal:** CLT art. 193
- **Incidência:** INSS ✅ | FGTS ✅ | IRRF ✅
- **No Gypsy:** boolean por função. Default true para eletricistas (NR-10)

```
adicional_periculosidade = salario_base × 0.30
```

## Insalubridade

- **Percentual:** 10% (leve), 20% (médio), 40% (máximo) sobre o **salário mínimo**
- **Base legal:** CLT art. 192
- **Incidência:** INSS ✅ | FGTS ✅ | IRRF ✅
- **No Gypsy:** enum por projeto (maioria das obras elétricas = nenhum ou leve)

```
SM_2025 = 1518.00
adicional_insalubridade = SM × grau_percentual  // 0, 0.10, 0.20, 0.40
```

> Periculosidade e insalubridade **não acumulam** (CLT art. 193 §2º). O trabalhador escolhe o mais vantajoso.

## Hora Extra

- **50%:** dias úteis (mínimo constitucional)
- **100%:** domingos e feriados (ou conforme convenção coletiva)
- **Incidência:** INSS ✅ | FGTS ✅ | IRRF ✅

```
hora_normal = salario_base / 220
custo_he_50  = custo_hh × 1.50
custo_he_100 = custo_hh × 2.00
```

> No Gypsy, HE não entra no custo/hora base. É modalidade separada no output do MC-1.
> O orçamentista escolhe a composição de horas (X normais + Y HE50% + Z HE100%) por assembly.

## Adicional Noturno

- **Percentual:** 20% sobre a hora normal
- **Horário:** 22h às 5h
- **Hora reduzida:** 52min30seg (simplificação gerencial: usar hora cheia)
- **Incidência:** INSS ✅ | FGTS ✅ | IRRF ✅

```
custo_noturno = custo_hh × 1.20
```

## EPI e Ferramentas

Não são adicionais CLT, mas são componentes do custo por função:

| Item | Tipo | Exemplo | Incidência |
|------|------|---------|------------|
| EPI (NR-6) | Custo empresa | R$ 180/mês (eletricista) | Não incide |
| Ferramentas individuais | Custo empresa | R$ 50/mês | Não incide |
| Exames médicos (NR-7) | Custo empresa | R$ 25/mês (rateio anual) | Não incide |
| Seguro de vida | Custo empresa | R$ 15/mês | Não incide |

Esses valores são **atributos da função** no BD-2 (`labor_roles.epi_monthly`, `tools_monthly`).
