# MC-001.1 — Encargos Sociais e Trabalhistas

## Estrutura dos encargos

Os encargos se dividem em 4 grupos (classificação SINAPI/TCU):

| Grupo | Descrição | Incide sobre |
|-------|-----------|-------------|
| **A** | Encargos sociais básicos | Remuneração bruta |
| **B** | Encargos trabalhistas (provisões) | Remuneração bruta |
| **C** | Incidência de A sobre B | Valores do grupo B |
| **D** | Encargos rescisórios | Remuneração bruta |

## Grupo A — Encargos Sociais Básicos

| Item | Horista (%) | Mensalista (%) | Base legal |
|------|-------------|----------------|------------|
| INSS Patronal | 20,00 | 20,00 | Lei 8.212/1991 |
| FGTS | 8,00 | 8,00 | Lei 8.036/1990 |
| RAT (construção civil, grau 3) | 3,00 | 3,00 | Variável por FAP (1,5%-6,0%) |
| SESI | 1,50 | 1,50 | |
| SENAI | 1,00 | 1,00 | |
| SEBRAE | 0,60 | 0,60 | |
| INCRA | 0,20 | 0,20 | |
| Salário Educação | 2,50 | 2,50 | |
| SECONCI (convenção coletiva) | 1,00 | 1,00 | Opcional, verificar enquadramento |
| **Total Grupo A** | **37,80** | **37,80** | |

> **Nota:** O RAT default é 3% (construção civil, grau 3). O FAP ajusta de 0,5× a 2,0×.
> Para orçamentação gerencial, usar 3% como default conservador.
> A desoneração (CPRB) reduz o INSS patronal gradualmente até 2028 — é parâmetro por projeto.

## Grupo C — Incidência de A sobre B

```
Grupo C = Grupo A × Grupo B
Horista:     37,80% × 39,40% = 14,89%
Mensalista:  37,80% × 20,83% = 7,87%
```

## Desoneração da Folha (CPRB) — Regime Transitório 2025-2028

Lei 14.973/2024 — reoneração gradual para construção civil:

| Ano | INSS Patronal sobre folha | CPRB sobre receita bruta |
|-----|---------------------------|--------------------------|
| 2024 (desoneração plena) | 0% | 4,5% |
| 2025 | 5% | 3,6% |
| 2026 | 10% | 2,7% |
| 2027 | 15% | 1,8% |
| 2028+ | 20% (regime normal) | — |

**No Gypsy:** desoneração é parâmetro por projeto. Default = regime normal (20%).

## Implementação

Arquivo: `src/data/social-charges.ts`

```typescript
export interface SocialChargeProfile {
  name: string;                    // "Lucro Real - Construção Civil"
  regime: 'lucro_real' | 'lucro_presumido' | 'simples' | 'mei';
  workerType: 'hourly' | 'monthly';
  groupA: { inssPatronal: number; fgts: number; rat: number; thirdParty: number; seconci: number };
  groupB: { thirteenthSalary: number; vacation: number; vacationBonus: number; dsr: number; sickLeave: number; licenses: number; justifiedAbsences: number };
  groupD: { noticePeriod: number; fgtsOnNotice: number; fgtsPenalty: number };
}
```

Os percentuais são **parâmetros configuráveis**, não constantes hardcoded.
Perfis pré-definidos (Lucro Real horista, mensalista, Simples, etc.) são seed no banco.
