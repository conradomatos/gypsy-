# Nomenclatura

- Inglês 100% para código (Python e TypeScript), banco de dados e API.
- Português (PT-BR) para UI, documentação em `docs/` e textos ao usuário final.
- Nome do produto: Gypsy ("CostAI" é branding antigo — só em docs
  arquivados/histórico).
- Em dúvida sobre nome de campo/tabela, pergunte ao Conrado — não invente.
