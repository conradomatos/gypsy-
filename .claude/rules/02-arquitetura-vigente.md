# Arquitetura vigente

DECIDIDO:
- Frontend: React + TypeScript + Vite; Tailwind CSS + shadcn/ui.
- Backend: Django + DRF; PostgreSQL; Python via uv.
- Integração: API REST.
- Engine de cálculo: Python puro isolado.
- Local-first.

REVOGADO (não usar nem recomendar como vigente):
- Coolify, Supabase, Edge Functions, Deno, RLS como arquitetura de autorização.

- **PROPOSTO/PENDENTE não vira DECIDIDO automaticamente.** Não adotar ferramenta
  ou biblioteca proposta como decisão sem aprovação do Conrado.
- Infraestrutura de produção é PENDENTE.

Detalhe em `01_ARQUITETURA/stack_tecnica.md` (produto), `toolchain.md`
(ferramentas) e `deploy_pipeline.md`. Não duplicar aqui.
