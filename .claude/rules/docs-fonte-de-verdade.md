# Docs são fonte de verdade

Antes de escrever código de um módulo, motor ou tabela, abra e siga o doc correspondente:

- Módulo → `docs/PROJETO_EXECUTIVO/GYPSY/04_MODULOS/M-00x...`
- Motor de cálculo → `docs/PROJETO_EXECUTIVO/GYPSY/05_MOTORES/MC-00x...`
- Tabela → `docs/PROJETO_EXECUTIVO/GYPSY/02_BANCO_DE_DADOS/BD-x...`
- Tela → `docs/PROJETO_EXECUTIVO/GYPSY/06_TELAS/...`

Divergiu do doc sem decisão registrada em `01_ARQUITETURA/registro_de_decisoes.md`?
**Pare e alinhe com o Conrado.** Não invente arquitetura, nome de tabela, regra ou fluxo.
